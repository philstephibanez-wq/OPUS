# OPUS License Intent

Status: license intent, pending legal validation before public commercial release.

## License profile

OPUS is not intended to be released as OSI open source under the current business decision.

The intended profile is:

```text
source-available
free for personal and non-commercial use
commercial use requires a paid commercial license
commercial use requires royalties
all rights reserved except permissions explicitly granted
```

## Free use scope

The intended free-use permission covers personal, private, educational, evaluation and non-commercial artistic use, subject to the final license text.

## Commercial use scope

Commercial use is not granted by the free-use profile.

Commercial use requires a signed or otherwise explicitly accepted commercial license with royalty terms defined by Philippe Stéphane Ibanez / Log&Play.

## Redistribution and forks

Unless the final license explicitly says otherwise, the free-use profile does not grant permission to redistribute OPUS, publish public forks, rebrand OPUS, remove copyright notices or sell OPUS as a competing product or service.

## Package inheritance

Official optional packages inherit this license intent unless their own manifest declares a stricter profile:

- OPUS core;
- OPUS_REF_BOOK;
- OPUS_USER_GUIDE;
- future official OPUS packages.

## Required files before public release

Before public commercial release, OPUS must ship with legally reviewed files:

```text
LICENSE
COMMERCIAL_LICENSE
NOTICE
COPYRIGHT
TRADEMARKS
PRIVACY
```

No package may advertise itself as OSI open source unless a future signed architecture decision explicitly changes the licensing model.
