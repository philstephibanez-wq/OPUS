# OPUS application packages

This directory contains source packages for official OPUS applications.

Rules:

- every official application package has its own `composer.json`;
- every official application package has its own `opus.application.json`;
- Composer installation is the official installation path;
- manual folder copy is not the official installation path.

Initial package sources:

- `opus-ref-book`
- `opus-demo`
- `opus-odbc-manager`
