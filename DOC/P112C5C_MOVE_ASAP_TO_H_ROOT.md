# P112C5C — ASAP à la racine de H:

## Rôle

Déplacer le framework ASAP hors de UwAmp/www/_packages vers une racine propre.

## Nouvelle source de vérité

H:\ASAP",
",


H:\UwAmp\www\ASAP -> H:\ASAP",
",


NO DOC CONTRACT, NO PATCH.

ASAP reste un framework mutualisable.

UwAmp expose un lien local pratique, mais la source de vérité est H:\ASAP.

## Nettoyage

H:\UwAmp\www\_packages\ASAP est supprimé par déplacement.

H:\UwAmp\www\_packages est supprimé seulement s’il est vide.
