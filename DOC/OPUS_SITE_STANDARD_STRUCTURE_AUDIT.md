# OPUS — Site standard structure audit

Contrat : `OPUS_SITE_STANDARD_CONTRACT_CORE`

## Règle

Tous les sites OPUS présents et futurs doivent respecter la structure standard `application` + `www/asset`.

## Sites détectés

- `opus-manager`
- `opus-p7-ops`

## Violations

`OPUS_SITE_STANDARD_STRUCTURE_VIOLATIONS_PRESENT`

- `sites/opus-manager/application`
- `sites/opus-manager/application/default`
- `sites/opus-manager/www`
- `sites/opus-manager/www/asset`
- `sites/opus-manager/www/asset/css`
- `sites/opus-manager/www/asset/js`
- `sites/opus-manager/www/asset/themes`
- `sites/opus-manager/src FORBIDDEN`
- `sites/opus-manager/public FORBIDDEN`
