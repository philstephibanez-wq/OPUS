## Plan de réalignement

1. Mapper chaque dérive vers une brique OPUS réelle.
2. Remplacer le code local par adapter OPUS explicite si nécessaire.
3. Supprimer les services locaux qui recréent le framework.
4. Remplacer HTML concaténé par templates/layouts OPUS.
5. Brancher navigation sur FSM/CL OPUS.
6. Brancher login/mot de passe sur Identity + ACL/RBAC OPUS.
7. Brancher langue sur OPUS I18N.
8. Ajouter un smoke OPUS-only bloquant.
