# skeleton

OPUS site/application scaffold generated by `composer opus:create-site`.

This is a professional starter site inspired by the historical ASAP modular demo model, without legacy esoteric sample content.

## Contract

- Application is modular.
- Common application resources live under `application/common`.
- Visible home rubrics are backed by declared modules/routes.
- Business modules live under `application/modules`.
- Public entry point only resolves declared routes and renders `.score` templates.
- HTML rendering must go through `.score` templates.
- External dependencies are forbidden unless an explicit ADR allows them.

Read `START_HERE.md` before modifying the site.
