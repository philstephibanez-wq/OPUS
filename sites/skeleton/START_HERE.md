# Start here

This site was generated by `composer opus:create-site`.

## Starter modules

- `Home` aggregates the declared rubrics.
- `Pages` demonstrates static page/content ownership.
- `Articles` demonstrates publication/article ownership.
- `Rubriques` demonstrates section/rubric ownership.
- `Documentation` demonstrates developer documentation ownership.

## First safe edits

1. Edit `resources/content/*.fr.json`.
2. Add another locale with `composer opus:add-language -- skeleton en --write`, then edit the generated localized content.
3. Edit the relevant module templates under `application/modules/<Module>/templates/pages/index.score`.
4. Add business modules with `composer opus:create-module -- skeleton ModuleName --write`.
5. Keep `public/index.php` as a starter front controller. Do not put page business logic there.
6. Keep HTML in `.score` templates.
7. Keep common assets/helpers/templates in `application/common`.
8. Put module-specific resources in `application/modules/<ModuleName>`.

## Rules

- No wild page creation.
- No module without a site.
- No page outside a module.
- No home card that is not backed by a module/route.
- No duplicated framework.
- No external dependency unless contractually approved.
